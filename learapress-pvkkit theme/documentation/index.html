Install Template

Step 1: Install Template Via FTP-Manager(File Zilla)

  1. Access your hosting server via an ftp client software like as FileZilla

if you face any problem during upload through FTP-Manager, please upload the Template via C-panel.

Alternative Step 2: Install Template Via C-panel

  1. Access your hosting server via c-panel
  2. Go to 'cpanel > file manager > public_html' Extract the template folder here
